## Installation

Verwenden Sie [Composer](/getcomposer.org/), um die Installation durchzuführen.

Führen Sie den folgenden Befehl im selben Verzeichnis wie Ihre Magento-Installation aus:

```
composer require omise/omise-magento
```
Wenn Sie das Modul zu einer bestehenden Magento-Installation hinzufügen, müssen Sie möglicherweise die folgenden Befehle ausführen, nachdem Sie Composer ausgeführt haben:
```
php bin/magento module:enable Omise_Payment --clear-static-content
php bin/magento setup:upgrade
php bin/magento setup:di:compile
```
## Konfiguration

Verbinden Sie Ihr Omise-Konto und aktivieren Sie Ihre bevorzugten Zahlungsmethoden.  
So ermöglichen Sie es Ihren Kunden, Zahlungen über Omise in Ihrem Shop vorzunehmen.

In Ihrem Magento-Admin-Bereich:

1. Gehen Sie zu `Stores` > `Configuration` > `Sales` > `Payment Methods`.
2. Die Konfigurationsoptionen werden unter dem Abschnitt **Omise** angezeigt.
3. Fügen Sie Ihre [Testschlüssel](https://sso-idp.omise.co/realms/engagement/protocol/openid-connect/auth?client_id=dashboard&redirect_uri=https%3A%2F%2Fdashboard.omise.co%2Fv2&state=6ec07f63-85db-4d93-be13-bb75ca934bee&response_mode=fragment&response_type=code&scope=openid&nonce=d13c716a-29c6-4199-8650-4b1de413cbe5&ui_locales=en&code_challenge=4Doxs1FG_nf7vkAZxWvECjIlQ9r97Q0T5FA9Iijzx_M&code_challenge_method=S256) und [Liveschlüssel](https://sso-idp.omise.co/realms/engagement/protocol/openid-connect/auth?client_id=dashboard&redirect_uri=https%3A%2F%2Fdashboard.omise.co%2Fv2&state=4c173f49-d4b4-48ba-95f4-13b70a21eaaa&response_mode=fragment&response_type=code&scope=openid&nonce=52d7109e-5bb9-4739-8ce0-67aaaa0d057a&ui_locales=en&code_challenge=NY8CciI7hPbFrW9G0IXNf_tIkQ8KODdO-VKUOEa4HS0&code_challenge_method=S256) hinzu und aktivieren Sie das Modul.

![configuration](/img/configuration.png)

> Hinweis: Wenn die Einstellung `Sandbox` auf `Yes` gesetzt ist, werden Ihre **Testschlüssel** verwendet.  
> Wenn die Einstellung `Sandbox` auf `No` gesetzt ist, werden Ihre **Liveschlüssel** verwendet.

## 3DS aktivieren

Wenn Sie 3DS-Zahlungen (auch bekannt als *3-D Secure*) verarbeiten möchten,  
[kontaktieren Sie bitte den Support](mailto:support@omise.co?subject=Feature%20Request%3A%20Magento), um 3DS in Ihrem Konto zu aktivieren.

> Wenn das Omise-Team 3DS in Ihrem Konto aktiviert hat, das Plugin jedoch noch nicht im Shop aktiv ist,  
> kann es zu Problemen bei der Zahlungsabwicklung kommen.  
> Sie erhalten dann eine Benachrichtigung mit dem Hinweis:  
> „3d secure is requested, but return URI is not set“.

## Upgrade auf das neue sichere Formular

Um auf das neue sichere Formular in Magento zu aktualisieren, folgen Sie den folgenden Anweisungen:

Verwenden Sie **Composer**, um Magento zu aktualisieren.

Führen Sie den folgenden Befehl im gleichen Verzeichnis wie Ihre bestehende Magento-Installation aus.

```
composer update omise/omise-magento

```
Führen Sie anschließend die folgenden Befehle aus, um Omise zu aktivieren:

```
php bin/magento module:enable Omise_Payment --clear-static-content
php bin/magento setup:upgrade
php bin/magento setup:di:compile
```
